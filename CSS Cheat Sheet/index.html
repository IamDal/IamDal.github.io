The Cheat Sheet

Like in most coding languages, css follows syntax and rules that define the structure of the language. Before one can understand the structure, it is essential to know the different types of elements and how they work. The table below is a basic introduction into the most commonly utilized css elements.

NameSyntaxDescription
element selector"main","div"Selects the html elements main and div by name to apply formatting to
element.class"main.blue-box"Selects the main element with the class "blue-box"
class selector".main",".blue-box"Selects all elements that have been assigned the named class to apply formatting
ID selector"#main","#id"Selects a specific element by it`s given ID
Select all "*" Selects all elements

Notes

We can combine element,class and id selectors to be more specific when styling.

  • Separating element selectors by a comma will select multiple elements.
    eg "main,div,p" will style those elements
  • separating elements selectors with a space will style element two within element 1.
    eg "main div" will style the div element, inside main

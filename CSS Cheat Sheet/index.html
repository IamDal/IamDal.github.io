CSS Selectors

What are selectors?

In order to add a style to our web page, we need a way to tell the computer what styles to apply to each element. Selectors are the name of the HTML element chosen to be stylized.

Selector Types

There are many different types of selectors. The ones described above apply to the html tag name it uses but there are many other ways to customize our selections
